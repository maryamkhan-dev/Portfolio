# Maryam Khan — Portfolio

Personal portfolio website built with React, Framer Motion, and Tailwind CSS.

**Live:** [https://portfolio-opal-eight-60.vercel.app/](https://portfolio-opal-eight-60.vercel.app/)

---

## Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- FormSubmit (contact form)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
